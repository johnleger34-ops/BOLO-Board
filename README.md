# BOLO Board v1.5

Native offline Android shift calendar and paycheck estimator for separate John and Alexis profiles.

## v1.5 changes
- Saved pay-stub history for each profile, newest to oldest.
- View and re-share old stubs without recalculating them from newer settings.
- Correct pay date and pay-period dates on generated images.
- Federal qualifying-child and other-dependent counts are converted automatically into withholding credits.
- Louisiana dependent/exemption count is included in estimated state withholding.
- Multiple same-day entries, full-moon night officer, health deduction scheduling, independent profiles, and calendar holidays remain included.
